# AI UX-интервьюер

Веб-инструмент для глубинного интервью с помощью LLM. Задаёт один вопрос, уточняет поверхностные ответы и завершает разговор при развёрнутом ответе.

## Стек

- **Backend**: Python + FastAPI + OpenAI API (streaming SSE)
- **Frontend**: React + TypeScript + Vite + Tailwind CSS + Framer Motion
- **Зависимости**: uv (Python), npm (Node)

## Запуск

### 1. Backend

```bash
cd backend
cp .env.example .env
# вставьте ваш OPENAI_API_KEY в .env

uv run uvicorn main:app --reload
# → http://localhost:8000
```

### 2. Frontend

```bash
cd frontend
npm install
npm run dev
# → http://localhost:5173
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

## Решение по системному промпту

Промпт написан от лица **UX-исследователя**, а не нейтрального бота — это даёт более естественный тон.

Ключевые правила в промпте:
1. **Один вопрос за раз** — модель не заваливает несколькими вопросами сразу.
2. **Чёткий критерий "подробности"** — конкретика: критерии выбора, сомнения, сравнение вариантов, эмоции. Это снижает вероятность преждевременного завершения.
3. **Запрет на раскрытие правил** — пользователь не видит инструкцию, разговор остаётся органичным.
4. **Тёплый тон** — без формализма, чтобы снизить барьер откровенности.

Почему `gpt-4o-mini`: быстрый, дешёвый, отлично следует инструкциям на русском языке. Streaming позволяет показывать ответ по мере генерации — нет ощущения "зависания".
